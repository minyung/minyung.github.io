# CashForm 개인정보 처리방침

본 앱은 사용자의 데이터를 안전하게 보호하기 위해 최선을 다합니다. 
사용자의 데이터는 선택하신 모드(로컬/공유) 및 기능 이용 여부에 따라 다음과 같이 처리됩니다.

### 1. 데이터 처리 원칙 및 모드별 운영
- **로컬 가계부 모드**: 입력하신 모든 데이터는 사용자의 기기 내부에만 저장됩니다. 단, 'AI 피드백' 기능을 이용할 경우에 한하여, 해당 지출 내역은 분석을 위해 일시적으로 서버로 전송되나 분석 즉시 파기되며 절대 저장되지 않습니다.
- **공유 가계부 모드**: 가계부 멤버 간 데이터 동기화를 위해 Google Cloud(Firestore) 서버에 데이터를 저장합니다.

### 2. 수집 및 전송하는 데이터 항목
- **로그인 정보**: 이메일 주소, Firebase Auth ID (UID)
- **가계부 데이터**: 지출/수입 내역, 카테고리, 메모
- **AI 분석 데이터 (전송 후 파기)**: AI 피드백 요청 시 해당 지출의 항목명, 금액, 메모
- **AI 피드백 결과 (저장)**: AI가 생성한 응답 메시지 (가계부 기록 및 멤버 공유 목적)

### 3. AI 서비스 이용 시 데이터 보호 (Google Vertex AI)
본 앱의 AI 피드백 기능은 기업용 **Google Cloud Vertex AI**를 통해 제공되며, 유저의 프라이버시를 위해 다음 정책을 엄격히 준수합니다.
- **비학습 원칙**: AI 분석을 위해 전송된 데이터는 Google의 기본 모델 개선이나 AI 학습 용도로 절대 활용되지 않습니다.
- **즉시 파기**: 분석을 위해 전송된 지출 데이터는 분석 결과 생성 직후 서버에서 즉시 소멸되며, 개발자나 제3자가 접근할 수 없습니다.
- **암호화 전송**: 모든 데이터는 표준 암호화 프로토콜을 통해 안전하게 전송됩니다.

### 4. 개인정보의 보유 및 이용 기간
- 사용자가 앱 내에서 '계정 삭제'를 요청하거나 서비스를 탈퇴할 때까지 보유합니다. 
- 탈퇴 시 Firestore에 저장된 모든 데이터(로그인 정보, 가계부 데이터, AI 피드백 결과)는 즉시 삭제됩니다.

### 4. 데이터 저장 및 위탁
- 본 서비스는 **Google Cloud (Firebase Firestore)**를 사용하여 데이터를 저장합니다.
