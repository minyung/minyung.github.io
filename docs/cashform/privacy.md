# CashForm 개인정보 처리방침

본 앱은 사용자의 데이터를 안전하게 보호하기 위해 최선을 다합니다.
본 앱의 '로컬 가계부' 모드 이용 시 입력하는 데이터는 사용자의 기기 내부에만 저장될 뿐, 개발자나 외부 서버로 전송 또는 수집되지 않습니다.

### 1. 수집하는 개인정보 항목
- **로그인 정보**: 이메일 주소, Firebase Auth ID (UID)
- **가계부 데이터**: 사용자가 직접 입력한 지출/수입 트랜잭션 내역, 카테고리 정보
- **기기 정보**: 푸시 알림 수신을 위한 기기 토큰 (선택 시)

### 2. 개인정보의 이용 목적
- **서비스 제공**: 사용자의 가계부 데이터를 클라우드(Firestore)에 동기화하여 기기 간 데이터 유지
- **공유 가계부**: Auth ID를 기반으로 다른 사용자 본인 확인 및 가계부 공유 기능 수행

### 3. 개인정보의 보유 및 이용 기간
- 사용자가 앱 내에서 '계정 삭제'를 요청하거나 서비스를 탈퇴할 때까지 보유합니다. 
- 탈퇴 시 Firestore에 저장된 모든 데이터는 즉시 삭제됩니다.

### 4. 데이터 저장 및 위탁
- 본 서비스는 **Google Cloud (Firebase Firestore)**를 사용하여 데이터를 저장합니다.
